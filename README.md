# Ajay Krishnan G - Personal Portfolio Website

Welcome to the repository for my personal portfolio website! This project serves as a digital resume and showcase of my professional journey, technical skills, and projects as a Full Stack Developer with a strong focus on Backend Engineering and DevOps.

## 👨‍💻 About Me
I am a Full Stack Developer with over 2 years of experience building scalable APIs, optimized databases, and robust server infrastructure. I specialize in shipping clean, maintainable solutions using technologies like Laravel, Node.js, Python, and more.

## 🛠️ Tech Stack & Skills
- **Backend & Frameworks**: Laravel / PHP, Yii2, Node.js / Express, Spring Boot / Java
- **Architecture & Design**: REST APIs, MVC Pattern
- **Database & DevOps**: MySQL, Linux / Server Administration, AWS, Git / Bitbucket, Python
- **Frontend & UI**: HTML5, Vanilla CSS3 (Custom properties, neon UI, animated grids), Vanilla JavaScript

## 📁 Portfolio Sections
The website is structured as a responsive single-page application containing the following sections:

1. **Hero**: A modern terminal-inspired introduction featuring a typing effect, high-level summary, and call-to-action buttons.
2. **Skills**: A detailed breakdown of my technical proficiencies categorized into "Backend & Frameworks" and "Database, DevOps & Tools" with visual progress bars.
3. **Experience**: An interactive timeline detailing my professional history, including my current role as a Full Stack Developer at Saiha Software Technologies and my past freelance experience.
4. **Education**: Academic background showcasing my Master of Science in Computer Science with Data Analytics and my Bachelor of Computer Application.
5. **Projects**: A dynamic, filterable grid of my featured work spanning across Full Stack, AI/ML, Backend, and Mobile/Health categories. Featured projects include:
   - **Career Compass V2**: AI-powered career guidance platform (React, Node.js, MySQL).
   - **Green Track**: Smart Waste Management AI system predicting complaint severities (Laravel, Python/Flask, Scikit-learn, NLP).
   - **Fall Detection & Health Monitor**: Real-time AI health monitoring (Node.js, Python, WebSockets).
   - **Spring Boot E-Commerce**: Enterprise Java backend for an e-commerce platform.
6. **GitHub**: A live GitHub contribution heatmap and statistics integration fetching real-time data from a custom backend API.
7. **Contact**: Links to reach out via Email, LinkedIn, and GitHub.

## 🚀 Website Features
- **Fully Responsive Design**: Ensures a seamless experience across mobile, tablet, and desktop screens.
- **Cyberpunk / Terminal Aesthetics**: Custom CSS featuring neon glows, scanline overlays, and an animated HTML5 canvas grid background.
- **Interactive UI**: Smooth scrolling navigation, category-based project filtering, scroll-to-top functionality, and subtle hover animations.
- **Live GitHub Heatmap**: Dynamically fetches and renders my GitHub contribution graph and repository statistics.

## 💻 Local Development
To view or modify this project locally:

1. Clone this repository to your local machine.
2. Simply open `index.html` in your web browser (or serve it using a local server extension like Live Server in VS Code).
3. *Note: The GitHub heatmap relies on the `portfolio_backend` API. Ensure the backend is running or the CORS settings allow your local environment if you are testing the GitHub integration locally.*
